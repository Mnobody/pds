# INSTALLATION

Install Git Large File Storage (lfs) if not installed: https://github.com/git-lfs/git-lfs/wiki/Installation

If something went wrong and your folder data is empty:
```sh
    git lfs pull
```

```sh
    docker compose up -d
```
